home > activities > manhattan gps watch

Custom building an analog watch (as close to my F-91W as possible) with GPS to show me the cross streets in Manhattan on which I am currently located. For example: if I'm at the Crunch on 54th between 7th and 8th (a godforsaken place—don't go here), it should say "54 x 7.5". If I'm getting a fish in Chelsea it should say "15.5 x 9.5". And if I'm in Brooklyn it should say "the suburbs".

While the goal is for the software to eventually make it onto a wristwatch, for now, the latest version will live here on this page.
